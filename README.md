# Visualizador de arquivos TGA 🌄

**Um visualizador simples e prático de imagens no formato TGA (.tga)** desenvolvido em C#.

### ✨ Funcionalidades

- **Abertura de arquivos TGA** — Suporte direto ao formato TGA
- **Navegação por setas** — Use as teclas ← (esquerda) e → (direita) para navegar entre as imagens da pasta
- **Interface limpa e leve** — Focado em praticidade
- **Visualização rápida** — Ideal para ver sequências de imagens TGA (muito útil em desenvolvimento de jogos, assets, etc.)

### 🎯 Sobre o Projeto

Este é um visualizador especializado no formato **TGA** (Truevision Graphics Adapter), bastante utilizado em jogos, engines e edição de texturas. O programa permite abrir uma imagem e navegar facilmente por todas as imagens TGA da mesma pasta usando apenas as setas do teclado.

### 🛠️ Tecnologias Utilizadas

- **C#** + **.NET**
- **Windows Forms**
- Leitura nativa de arquivos TGA (sem dependência de bibliotecas externas pesadas)
